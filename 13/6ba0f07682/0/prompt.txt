## Task 3: Stripe Integration

### Migration

**`apps/api/prisma/schema.prisma`** — thêm Stripe fields vào User model:
```prisma
stripeCustomerId     String?  @unique @map("stripe_customer_id")
stripeSubscriptionId String?  @map("stripe_subscription_id")
stripePriceId        String?  @map("stripe_price_id")
```
- Giữ `Site.plan` như denormalized copy (analytics queries đọc site.plan, không cần thay đổi)
- Migration: `npx prisma migrate dev --name add_stripe_fields_to_user`

### Backend (NEW files)

```
apps/api/src/billing/
├── billing.module.ts
├── billing.service.ts
├── billing.controller.ts
└── dto/create-checkout.dto.ts
```

**Endpoints:**
- `POST /billing/checkout` (JWT) — tạo Stripe Checkout Session, trả `checkoutUrl`
- `POST /billing/webhook` (public, raw body) — handle subscription events, update User.plan + Site.plan
- `GET /billing/portal` (JWT) — trả Stripe Billing Portal URL

**billing.service.ts:**
- `createCheckoutSession(userId, plan)` — tạo/lấy Stripe customer, tạo checkout session
- `handleWebhook(rawBody, signature)` — verify + route events
- `handleSubscriptionUpdated(subscription)` — map priceId → PlanType, update User + all Sites trong transaction
- `handleSubscriptionDeleted(subscription)` — reset về 'free'
- `getBillingPortalUrl(userId)` — tạo portal session

**Raw body cho webhook:** Configure trong `main.ts` dùng Fastify `addContentTypeParser`

**`apps/api/src/app.module.ts`** — register BillingModule
